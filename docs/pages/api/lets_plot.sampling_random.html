    lets_plot.sampling_random¶

    lets_plot.sampling_random(n, seed=None)¶

    Return a subset of randomly selected items.

    Parameters

    • n (int) – Number of items to return.

    • seed (int) – Number used to initialize a pseudo random number generator.

    Returns

    Random sample specification.

    Return type

    FeatureSpec

    import numpy as np
from lets_plot import *
LetsPlot.setup_html()
np.random.seed(27)
mean = np.zeros(2)
cov = [[.9, -.6],
       [-.6, .9]]
x, y = np.random.multivariate_normal(mean, cov, 10000).T
ggplot({'x': x, 'y': y}, aes(x='x', y='y')) + \
    geom_point(sampling=sampling_random(1000, 35))
