    lets_plot.maptiles_zxy¶

    lets_plot.maptiles_zxy(url: str, attribution: Optional[str] = None, min_zoom: Optional[int] = None, max_zoom: Optional[int] = None, **other_args) → dict¶

    Makes raster tiles config. Can be used individually in geom_livemap() or in every livemap via LetsPlot.set(…)

    Parameters

    • url (string) – Template for a standard raster ZXY tile provider with {z}, {x} and {y} wildcards, e.g. ‘http://my.tile.com/{z}/{x}/{y}.png’

    • attribution (string) – An attribution or a copyright notice to display on the map as required by the tile license. Supports HTML links <a href=”http://www.example.com”>Example</a>

    • min_zoom (int) – Minimal zoom limit

    • max_zoom (int) – Maximal zoom limit

    • other_args (**string) – Any key-value pairs that can be substituted into the url template

      maptiles_zxy(

      url = ‘http://maps.example.com/{z}/{x}/{y}.png?access_key={key}’ key = ‘MY_ACCESS_KEY’

      )

    Returns

    Return type

    Tile provider settings
