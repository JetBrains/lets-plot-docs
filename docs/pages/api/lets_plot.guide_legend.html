    lets_plot.guide_legend¶

    lets_plot.guide_legend(nrow=None, ncol=None, byrow=None)¶

    Legend guide.

    Parameters

    • nrow (int) – Number of rows in legend’s guide.

    • ncol (int) – Number of columns in legend’s guide.

    • byrow (bool, default=True) – Type of output: by row, or by column.

    Returns

    Color guide specification.

    Return type

    FeatureSpec

    Note

    Legend type guide shows key (i.e., geoms) mapped onto values.

    import numpy as np
from lets_plot import *
LetsPlot.setup_html()
n = 100
np.random.seed(42)
x = np.random.uniform(size=n)
y = np.random.uniform(size=n)
c = np.random.choice(list('abcdefgh'), size=n)
ggplot({'x': x, 'y': y, 'c': c}, aes('x', 'y')) + \
    geom_point(aes(shape='c'), size=4, alpha=.7) + \
    scale_shape(guide=guide_legend(nrow=3))
