    lets_plot.ggsave¶

    lets_plot.ggsave(plot: Union[lets_plot.plot.core.PlotSpec, lets_plot.plot.plot.GGBunch], filename: str, *, path: Optional[str] = None, iframe: bool = True) → str¶

    Exports plot or “bunch” to a file. Supported formats: SVG, HTML.

    The exported file is created in directory ${user.dir}/lets-plot-images if not specified otherwise (see the path parameter).

    Parameters

    • plot (PlotSpec or GGBunch) – Plot specification to export.

    • filename (str) – The name of file. It must end with a file extension corresponding to one of the supported formats: svg, html (or htm).

    • path (str) – Path to a directory to save image files in. By default it is ${user.dir}/lets-plot-images.

    • iframe (bool, default=True) – Whether to wrap HTML page into a iFrame. Only applicable when exporting to HTML.

    Returns

    Absolute pathname of created file.

    Return type

    str
